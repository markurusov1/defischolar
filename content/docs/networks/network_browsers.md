# Network Browsers
{{% hint info %}}
- Remember how we said that blockchain is a completely transparent system? 
- This site - [Blockchain Explorer - Bitcoin Tracker & More | Blockchain.com](https://www.blockchain.com/explorer) - 
  lets you see all transactions in three major L1 networks 
- As an example, [this link](https://www.blockchain.com/explorer/blocks/btc/920900)-lets you understand what 
  happened to a specific block
- This is [Ethereum network browser](https://etherscan.io/)
- Keep these links handy, even if you don’t immediately understand what’s going on
  {{% /hint%}}
